# 🧮 Advanced Scientific Calculator

A modern and interactive scientific calculator built using **HTML, CSS, and Vanilla JavaScript**.  
This project features a clean UI, advanced mathematical operations, and enhanced user experience.

---

## 🚀 Features

- ➕ Basic arithmetic operations (+, −, ×, ÷)
- 📐 Brackets support ( )
- 🔢 Power operator (^)
- 🧮 Factorial (!)
- 🟡 Pi (π) support with auto multiplication
- 📊 Scientific functions:
  - sin()
  - cos()
  - log()
  - √ (square root)
- 🌗 Dark/Light theme toggle
- ✨ Glowing modern UI design
- ⌨️ Keyboard input support
- 🕘 History panel:
  - Click to reuse result
  - Delete individual entries
  - Clear all history
